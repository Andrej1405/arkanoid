# Arkanoid

The basic implementation of the `Arkanoid` game.
All shapes are drawn using standard canvas methods. The project is under development.

Run the command to build the project:
```
npm run build
```

Run to start server:
```
npm run start
```

And go to `http://localhost:8080/`
